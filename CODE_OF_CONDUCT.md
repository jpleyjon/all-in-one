# Code of Conduct

We are committed to making this project a welcoming and respectful place for everyone.

## Our Pledge

We, as contributors and maintainers, pledge to make participation in this project
harassment-free for everyone, regardless of background, identity, or experience.

## Expected Behavior

- Be respectful and professional in all project spaces.
- Critique ideas and code; avoid personal attacks.
- Communicate clearly, with patience and empathy.
- Respect others' boundaries, time, and preferred communication styles.
- Keep project channels focused and constructive.

## Unacceptable Behavior

- Harassment, threats, or intimidation.
- Hate speech, discriminatory language, or slurs.
- Sexual, political, or personal attacks.
- Trolling, spam, or repeated disruption.
- Doxxing or sharing private information without consent.

## Scope

This code applies in all project spaces, including issues, pull requests, discussions,
chat, and any official project channel where someone is representing this project.

## Reporting

If you experience or witness harmful behavior, please report it through the
project’s GitHub channels:
- Open a GitHub issue and mark it as a conduct concern.
- If your report is sensitive, use the repository’s private reporting option when available.

## How reports are handled

This is a small project with limited moderation bandwidth. Reports are reviewed when
possible and handled on a case-by-case basis with the goal of keeping the project
safe and collaborative.

Responses may include:

- Asking participants to adjust behavior.
- Temporarily or permanently removing offending comments.
- Removing contributions that do not meet the community standards.
- Blocking repeat offenders from project participation when needed.

## Attribution

Based on common open-source community conduct standards.
